# 收藏夾

# 打開收藏窗口

  1. 滑鼠移動到 收藏圖示 上。

  2. 點擊 收藏圖示。

  3. 你手動收藏的提示詞，都會被記錄在收藏夾中，除非你手動取消收藏，否則它們將會永久保留。

# 功能說明

  1. 設置名稱，輸入此條收藏記錄的名稱，按回車鍵保存。

  2. 取消收藏，點擊後，會刪除你選中的收藏記錄（不可恢復）。

  3. 複製，點擊後，會將你選中的收藏中的Prompts複製到你的剪貼板。

  4. 使用，點擊後，會清空你已經輸入的Prompts，並將你選中的收藏中的Prompts填充到輸入框中。
